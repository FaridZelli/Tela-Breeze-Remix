# Tela Breeze Remix
My lightweight fork of Tela for personal use
## Installation:
- Get the latest [release](https://github.com/FaridZelli/Tela-Breeze-Remix/releases/latest)
- Extract Tela-Breeze-Remix to `/usr/share/icons` or `~/.local/share/icons`
